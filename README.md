EcoWild-Manager

Por: Ana Isabel Loaiza Pacheco.
Fecha: 28/6/2026

Objetivo del sistema:
Desarrollar un sistema de gestión para EcoWild Centro Veterinario que permita registrar y consultar información de mascotas, propietarios y citas médicas, aplicando los conocimientos adquiridos durante la asignatura.

Funcionalidades
El sistema permite:

- Registrar mascotas.
- Registrar propietarios.
- Registrar citas veterinarias.
- Consultar la información registrada.
- Mostrar los registros almacenados.
- Salir del sistema mediante un menú interactivo.

Tecnologías utilizadas:
- Python
- Visual Studio Code
- GitHub
